This page retrieves data from the BFF official site and uses it to draw graphs.

How to use
  1. First search players that may interest you. They will be added in the players pool
  2. Then select / unselect players in the players pool by clicking on them
  3. Finally create a chart with the selected players. The chart wont be created if more than 10 players are selected
